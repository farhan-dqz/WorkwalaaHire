# Website Pages

This folder contains main website pages for Workwalaa.

## Files Included

### Core Pages
- `aboutus.html` - About Workwalaa company page
- `careers.html` - Careers and job opportunities page
- `blogs.html` - Blog and articles page
- `interview-prep.html` - Interview preparation services page

## Purpose
These are the main informational and service pages of the Workwalaa website, providing comprehensive information about the company, services, and resources.

## Features
- Company information and history
- Career opportunities and job listings
- Educational blog content
- Interview preparation resources

## Usage
All files maintain their original UI and backend functionality. They are organized here for better project structure and maintainability.
