# Workwalaa Project Structure

This document outlines the organized folder structure of the Workwalaa project.

## 📁 Folder Organization

### 🏠 Root Directory
- `index.html` - Main homepage with testimonials slider
- `assets/` - Static assets (images, styles, scripts)
- `assist/` - Workwalaa Assist module
- `contact/` - Contact form and communication module
- `testimonials/` - Testimonials section module

### 📂 New Organized Folders

#### 🎯 `hiring/` - Hiring & Recruitment Services
All hiring-related pages organized by type:
- **Employee Hiring**: hire-employees.html, employee-hiring-service.html, worker-hiring-platform.html
- **Location-Based**: hire-workers-in-kerala.html, hire-workers-in-kochi.html, hiring-partner-in-delhi.html
- **General Hiring**: hire-workers.html, hiring-company.html, hiring-partner.html
- **Recruitment**: recruitment-partner.html, staffing-agency-in-pune.html, staffing-partner.html

#### 💼 `consultancy/` - Job Consultancy Services
Professional consultancy and guidance pages:
- **General**: job-consultancy.html, recruitment-consultancy.html
- **Location-Based**: job-consultancy-in-malappuram.html, recruitment-consultancy-in-chennai.html

#### 📄 `pages/` - Main Website Pages
Core informational pages:
- aboutus.html - Company information
- careers.html - Career opportunities
- blogs.html - Blog and articles
- interview-prep.html - Interview preparation

#### ⚙️ `services/` - Service Components
Service-oriented pages and components (currently prepared for future use)
