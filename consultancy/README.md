# Consultancy Services

This folder contains all job consultancy and recruitment consultancy pages for Workwalaa.

## Files Included

### General Consultancy
- `job-consultancy.html` - Main job consultancy services
- `recruitment-consultancy.html` - Recruitment consultancy solutions

### Location-Based Consultancy
- `job-consultancy-in-malappuram.html` - Job consultancy in Malappuram
- `recruitment-consultancy-in-chennai.html` - Recruitment consultancy in Chennai

## Purpose
These pages provide professional consultancy services for job seekers and employers, offering guidance and support throughout the recruitment process.

## Features
- Location-specific consultancy services
- Job search assistance
- Recruitment process guidance
- Professional career consulting

## Usage
All files maintain their original UI and backend functionality. They are organized here for better project structure and maintainability.
