# Hiring Services

This folder contains all hiring and recruitment related pages for Workwalaa.

## Files Included

### Employee Hiring
- `hire-employees.html` - Main employee hiring page
- `employee-hiring-service.html` - Professional employee hiring services
- `worker-hiring-platform.html` - Worker hiring platform page

### Location-Based Hiring
- `hire-workers-in-kerala.html` - Hire workers in Kerala
- `hire-workers-in-kochi.html` - Hire workers in Kochi
- `hiring-partner-in-delhi.html` - Hiring partner in Delhi

### Recruitment Services
- `hire-workers.html` - General worker hiring
- `hiring-company.html` - Company hiring services
- `hiring-partner.html` - Hiring partner program
- `recruitment-partner.html` - Recruitment partnership
- `staffing-agency-in-pune.html` - Staffing services in Pune
- `staffing-partner.html` - Staffing partner program

## Purpose
These pages provide comprehensive hiring solutions for businesses looking to recruit qualified employees across different locations and job categories.

## Features
- Location-specific hiring pages
- Industry-specific recruitment
- Partner program integration
- Professional staffing solutions

## Usage
All files maintain their original UI and backend functionality. They are organized here for better project structure and maintainability.
